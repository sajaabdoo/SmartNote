# SmartNote 📝

SmartNote is a professional notes application built with Flask (Python).  
It allows users to add, edit, delete, and search notes with live highlights.  
Notes are sorted by priority and include word count and timestamps.

---

## Features

- Add new notes
- Edit existing notes
- Delete notes
- Search notes with highlight
- Sort notes by priority: High 🔴, Medium 🟡, Low 🟢
- Word count and date tracking
- Clean and responsive UI
